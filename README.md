# Capstone 2: Business Analysis with Python

## Project Overview

The project analyzes sales performance data for EmporiUm, a student bookstore chain that sells books, technology gear, art supplies, and other educational products. The company operates across multiple regions, with each region divided into sales territories managed by individual territory managers. This analysis compares sales performance between 2 south region territories managed by Len Jensen and Lana Ilana

---

## Objectives

The goals of this project is to:

* Identify territory managers and store locations
* Analyze monthly sales revenue by territory
* Rank store sales performance
* Identify top customers using rewards membership data
* Analyze product category sales and transaction trends
* Create data visualizations using Matplotlib
* Provide recommendations for future marketing efforts
